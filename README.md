# Berlines Intelligence
This is a repository for the project "Berliner Intelligence".

## Notebooks
* Analysis: Contains a linear regression model for advanced analytics
* Schema: Contains the code to create the needed Tables for the transition from OLTP to OLAP
* Transform: Contains multiple Data-Preparation-Steps to enable the analysis

## Datasets
* Apple Product Price List: Contains the prices from 26 Country for several Apple Products
* Data: Contains information about inequalities and other human related values
* Exchange Rate: Contains the exchange rates for several currencies to USD

## Python-Files
* Hilfsfunktionen (Contributed through Prof. Sebastian Fischer): Contains functions to work with SQL through Python
